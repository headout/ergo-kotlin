    Changelog¶

    All notable changes to this project will be documented in this file. See standard-version for commit guidelines.

    1.2.0 (2020-09-24)¶

    Features¶

    • add immediate respond result handler (80daf60)
    • add jobParser to job result handlers (f1350af)
    • added dokka plugin for doc generation (2b2845e)
    • generate deserializeJobResult in job parser (f62fc95)
    • modularize result handler to give flexibility to client (771cc93)

    Bug Fixes¶

    • fixed tests for sqs service with buffered results (6cf387a)

    1.1.5 (2020-09-24)¶

    Bug Fixes¶

    • use fixed threadpool and runJob without context switch (4d6f42e)
    • use IO dispatcher for ping message (48ae379)

    1.1.4 (2020-09-24)¶

    Bug Fixes¶

    • parse 404 results too (32af477)

    1.1.2 (2020-09-24)¶

    Bug Fixes¶
