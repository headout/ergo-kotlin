class InMemoryBufferJobResultHandler(maxResultsToBuffer: Int, timeoutCollectResult: Long) : JobResultHandler

Implements a in-memory buffer for executed job results before attempting to push all buffered results to SQS in bulk.

The buffer is grown till maxResultsToBuffer size before results are pushed and buffer is cleared. There's also a ticker timeout happening regularly with a delay of timeoutResultCollect which when received, pushes the buffered results and clears the buffer regardless of whether the buffer is full.

Parameters

maxResultsToBuffer

capacity of the buffer beyond which buffer is emptied and pushed (for SQS, recommended value is 10)

timeoutCollectResult

timeout interval (in milliseconds) to push and clear the buffer (default is 2 minutes)
